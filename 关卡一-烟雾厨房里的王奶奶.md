# 《红花救援队》关卡一正式策划案：烟雾厨房里的王奶奶

## 一、文档定位与全局口径

本文件用于定义《红花救援队》正式关卡一的教学目标、玩法规则、数值模型、流程、评分与开发实现边界。关卡一承担新手正式关的职责，需要让玩家第一次完整体验“判断安全 - 网格行动 - NPC沟通 - 护送撤离 - 复盘学习”的核心循环。

### 1. 全局设计原则

| 维度 | 统一口径 |
| --- | --- |
| 游戏模式 | 只设置“标准模式”，不做高压惩罚模式或复杂操作模式 |
| 基础玩法 | 网格救援 + 行动卡判断 + 关键节点小游戏 + AI复盘 |
| 失败处理 | 允许立即重试，并给出错误原因与正确流程复盘 |
| 公益优先级 | 正确安全原则高于通关效率，不能让危险行为成为最优解 |
| AI助手定位 | “花花”提供提示、反馈、复盘和知识卡，不自动完成核心操作 |
| 全龄体验 | 大按钮、清晰文字、明确图标、回合制推进，不因现实思考时间惩罚玩家 |

### 2. 统一回合与行动点

本关沿用全游戏基础网格口径：

| 项目 | 规则 |
| --- | --- |
| 1回合 | 玩家消耗完行动点并确认结束，或执行关键救援动作后进入环境结算 |
| 每回合AP | 4 AP |
| 移动1格 | 1 AP |
| 观察相邻区域 | 1 AP |
| 拾取/使用道具 | 1 AP |
| 安抚沟通 | 1 AP |
| 清除障碍 | 2 AP |
| 高风险动作 | 不作为推荐解法，只作为错误或低评分分支 |

### 3. 公益安全边界

本关表现的是“早期厨房险情 + 楼道近距离协助”，不是鼓励普通人冲入火场。玩家身份设定为社区“红花救援队”成员，接受过基础安全培训，且关卡场景中火势仍处于可控早期。

必须坚持的公益边界：

- 进入前必须判断自身安全，并呼叫119或通知周围成年人/物业协助。
- 如果火势失控、明火封路、浓烟风险过高，系统应判定为“等待专业救援”，而不是奖励继续深入。
- 关闭燃气、取防烟面罩等行为只在可安全接近时作为加分项，不作为鼓励冒险的唯一通关条件。
- 玩家救援的核心不是“英雄式冲进火场”，而是“判断风险、正确沟通、低姿防烟、尽快撤离”。

| 项目 | 内容 |

| 关卡编号 | 正式关卡一 |
| 关卡名称 | 烟雾厨房里的王奶奶 |
| 推荐时长 | 3-4分钟 |
| 推荐有效回合 | 7-9回合 |
| 最大救援窗口 | 12回合，超过后火场风险过高，进入失败复盘 |

## 三、教学目标与体验目标

### 1. 公益教学目标

玩家完成本关后，应形成以下认知：

| 教学点 | 玩家需要学会的内容 |
| --- | --- |
| 先保自身安全 | 先判断现场风险，无法安全进入时等待专业救援 |
| 及时呼救 | 发现火情后第一时间呼叫119或请求周围成年人协助 |
| 浓烟避险 | 低姿移动，避开明火和高浓度烟雾，不盲目深入 |
| 老人沟通 | 面对听力弱、恐慌或认知混乱的老人，应蹲下、慢说、说明身份 |
| 快速撤离 | 救援目标是离开危险环境，不在现场逗留或返回取物 |

### 2. 游戏体验目标

本关希望玩家感受到：

- 视野受限带来的探索紧张感。
- AP有限时“快救人”和“稳救人”的取舍。
- 王奶奶恐慌移动带来的动态救援压力。
- 成功安抚并护送老人离开后的情绪释放。

## 四、核心玩法循环

本关循环不是单纯找人，而是围绕“安全进入、有限探索、沟通稳定、撤离护送”展开。

1. 判断现场安全并触发呼救。
2. 在烟雾遮挡的网格中探索路线。
3. 根据AP决定是否拿防烟面罩、清障、关闭燃气。
4. 接近王奶奶并用正确沟通降低恐慌。
5. 引导王奶奶沿安全路线撤离。
6. 根据玩家行为生成评分、复盘和知识卡。

## 五、场景与地图设计

### 1. 地图尺寸

本关地图为 **10x8网格**，横向10格，纵向8格。

采用10x8的原因：

- 左右距离足够表现“楼道安全区 - 厨房入口 - 烟雾深处”的空间层次。
- 玩家需要2-4步接触第一个关键物件，5-7步有机会接近王奶奶。
- 8x8会让找人、清障、绕路、护送都被压缩，降低救援感。

### 2. 空间分区

| 区域 | 位置 | 功能 |
| --- | --- | --- |
| 安全区/楼道 | 左侧2列 | 起点与撤离终点，开局可见 |
| 厨房入口 | 中左区域 | 放置防烟面罩、第一处障碍、低浓度烟雾 |
| 厨房中段 | 中部区域 | 形成绕路、观察和清障选择 |
| 灶台/燃气区 | 右中区域 | 放置燃气阀门和高风险火源 |
| 烟雾深处 | 右侧区域 | 王奶奶初始活动区，视野受限 |

### 3. 主要元素

| 元素 | 设计说明 |
| --- | --- |
| 玩家起点 | 楼道与厨房交界处，开局可见 |
| 王奶奶 | 初始位于右侧烟雾区，安抚前会因恐慌小范围移动 |
| 安全区 | 地图最左侧楼道出口，王奶奶到达后完成关卡 |
| 防烟面罩 | 中左区域，相邻格被“翻倒的冰箱”阻挡 |
| 燃气阀门 | 灶台附近，相邻格被“倒下的储物架”阻挡 |
| 烟雾格 | 每回合对王奶奶造成生命状态损失，未探索时显示为未知 |
| 高浓度烟雾格 | 伤害更高，花花会提示绕行 |
| 火焰格 | 不可通行，尝试进入会被系统阻止并扣除少量评分 |
| 障碍物 | 消耗2 AP清除，可改变撤离路线 |

## 六、机制与数值模型

### 1. 初始数值

| 项目 | 数值 |
| --- | --- |
| 玩家每回合AP | 4 |
| 最大救援窗口 | 12回合 |
| 王奶奶生命状态 | 100 |
| 王奶奶恐慌值 | 60 |
| 王奶奶信任值 | 20 |
| 火场风险值 | 30/100 |
| 普通烟雾伤害 | 每回合 -5生命状态 |
| 高浓度烟雾伤害 | 每回合 -8生命状态 |
| 防烟面罩效果 | 王奶奶烟雾伤害降为每回合 -1 |
| 火势扩散 | 默认每2回合扩散1格 |
| 关闭燃气后 | 火势扩散改为每3回合1格，且火场风险立即 -15 |
| 失败条件 | 王奶奶生命状态归零，或火场风险达到100，或超过12回合 |

### 2. Fog of War

开局时，厨房内部除入口周围外全部被浓烟遮挡，显示为“未知”。

可视规则：

- 玩家进入格子后，解锁该格与相邻1格。
- 使用“观察”消耗1 AP，额外解锁指定方向2格。
- 拾取防烟面罩后，玩家观察范围 +1格。
- 王奶奶在未知区域移动时，只显示模糊轮廓或咳嗽提示，不直接暴露精确位置。

设计目的：

- 表现浓烟遮挡视线。
- 避免玩家开局直接算出最优路线。
- 让“观察”和“快速推进”形成AP取舍。

### 3. 王奶奶状态

| 状态 | 触发条件 | 游戏效果 |
| --- | --- | --- |
| 恐慌 | 初始状态 | 每次火势扩散时小范围移动1格 |
| 信任不足 | 信任值低于40 | 无法触发跟随撤离 |
| 已安抚 | 选择正确沟通后 | 恐慌值 -30，信任值 +30，解锁“引导撤离” |
| 已防护 | 给王奶奶使用防烟面罩 | 烟雾伤害大幅降低，信任值 +10 |
| 体力下降 | 生命状态低于40 | 移动跟随延迟增加，撤离难度提高 |

王奶奶随机移动需要被限制在合理范围内：

- 只在烟雾深处的活动区内移动。
- 不会主动走进火焰格。
- 每2回合最多移动1次。
- 移动前给玩家音效或轮廓提示，避免随机性显得不公平。

### 4. 清障与可选目标

| 可选目标 | 成本 | 收益 | 公益表达 |
| --- | --- | --- | --- |
| 拿防烟面罩 | 清障2 AP + 拾取1 AP | 降低护送期烟雾伤害，扩展视野 | 做好防护再救援 |
| 关闭燃气 | 清障2 AP + 关闭1 AP | 降低火势扩散与火场风险 | 只在安全接近时处理危险源 |
| 直接找王奶奶 | 节省AP | 更快接触NPC，但护送压力更大 | 快速救人可以成立，但不能忽略安全判断 |
| 多次观察 | 每次1 AP | 降低走错路概率 | 稳妥探索优于盲目深入 |

## 七、标准通关流程

### 阶段一：安全判断与呼救

开场剧情：

花花：“厨房里有浓烟，王奶奶可能还在里面。先看清现场，确认自己还能安全接近，再呼救，不要盲目冲进去。”

行动卡：

| 选项 | 结果 |
| --- | --- |
| 判断现场安全并呼叫119 | 正确，进入下一阶段，评分提高 |
| 直接冲进厨房 | 错误，火场风险 +10，花花提示先判断安全 |
| 回房间找贵重物品 | 错误，直接进入失败复盘 |
| 打开窗户大声喊 | 部分错误，可能扩大烟气流动，提示改为撤离与呼救 |

正确选择后，系统显示：

- AP：4
- 王奶奶生命状态：100
- 王奶奶恐慌值：60
- 火场风险值：30
- 当前回合：第1回合

### 阶段二：进入厨房入口并观察

玩家从楼道进入厨房入口附近。厨房内部大部分区域被浓烟遮挡。

可选行为：

- 移动探索。
- 消耗1 AP观察指定方向。
- 优先前往防烟面罩。
- 直接向咳嗽声方向接近王奶奶。

花花提示重点：

“烟很浓，低姿前进，优先找安全路线。看不清时先观察，不要直接穿过火焰和高浓度烟雾。”

### 阶段三：防烟面罩取舍

防烟面罩相邻格有“翻倒的冰箱”障碍。

玩家选择：

| 选择 | 成本 | 后果 |
| --- | --- | --- |
| 清障并拾取面罩 | 3 AP | 后续护送更安全，评分提高 |
| 暂时放弃面罩 | 0 AP | 更快接近王奶奶，但烟雾伤害压力更大 |
| 多次绕路寻找替代路线 | 2-4 AP | 可能节省清障，但时间风险增加 |

设计意图：

面罩是“稳救人”的代表，不是强制通关钥匙。玩家可以快救，但高分需要更完整的安全准备。

### 阶段四：燃气阀门可选处理

燃气阀门位于灶台附近，相邻格有“倒下的储物架”。

触发条件：

- 玩家已观察到阀门位置。
- 阀门附近无火焰格直接封锁。
- 火场风险值未达到80。

玩家选择：

| 选择 | 成本 | 后果 |
| --- | --- | --- |
| 清障并关闭燃气 | 3 AP | 火场风险 -15，火势扩散变慢，评分提高 |
| 放弃关闭，优先救人 | 0 AP | 合理选择，通关可行，但火势压力更大 |
| 在明火封锁时强行接近 | 不允许执行 | 花花提示等待专业救援 |

### 阶段五：接近并安抚王奶奶

玩家移动到王奶奶相邻格后触发对话：

王奶奶：“你是谁啊？我家门在哪里？我看不清……”

行动卡：

| 选项 | 结果 |
| --- | --- |
| 蹲下并慢慢说明身份 | 正确，恐慌值 -30，信任值 +30 |
| 大声催促她快走 | 恐慌值 +20，下一回合拒绝移动 |
| 直接拉她离开 | 恐慌值 +30，拒绝移动1回合，评分下降 |
| 让她自己找出口 | 生命状态持续下降，花花提示重新沟通 |

正确选择后解锁“引导撤离”。

如果玩家持有防烟面罩，可使用道具：

- 消耗1 AP。
- 王奶奶烟雾伤害降为每回合 -1。
- 信任值 +10。
- 评分提高。

### 阶段六：护送撤离

王奶奶被安抚后，进入护送阶段。

护送规则：

- 玩家移动1格，王奶奶跟随移动1格。
- 王奶奶不再随机移动。
- 经过烟雾格时，根据是否佩戴面罩结算伤害。
- 火焰格不可通行。
- 障碍物可消耗2 AP清除。
- 如果王奶奶生命状态低于40，跟随有1步延迟，需要玩家等待或选择更安全路线。

护送阶段的主要选择：

- 走短路，但经过更多烟雾。
- 走远路，但避开高浓度烟雾和火焰。
- 清除障碍，换取更稳定撤离路线。

### 阶段七：到达安全区与结算

当玩家和王奶奶都到达安全区，关卡完成。

结算展示：

- 是否先判断安全并呼救。
- 是否低姿避烟并合理观察。
- 是否成功安抚王奶奶。
- 是否使用防烟面罩。
- 是否安全关闭燃气。
- 王奶奶剩余生命状态。
- 用了多少回合。
- 是否发生高风险行为。

## 八、错误处理与教学反馈

| 错误行为 | 游戏后果 | 教学反馈 |
| --- | --- | --- |
| 不判断安全直接进入 | 火场风险 +10，评分下降 | 先判断自身安全，不能盲目进入危险现场 |
| 直接拉走王奶奶 | 恐慌值 +30，拒绝移动1回合 | 面对恐慌老人要先说明身份并稳定情绪 |
| 不安抚直接撤离 | 无法触发跟随 | 信任不足时，老人可能不配合移动 |
| 长时间不呼救 | 评分大幅下降，复盘重点提示 | 火情需要尽快通知119或成年人 |
| 不拿防烟面罩 | 护送压力增加，但可通关 | 快速救人可以成立，但高分需要安全防护 |
| 不关闭燃气 | 火势扩散更快 | 危险源处理是加分项，但前提是安全可接近 |
| 试图进入火焰格 | 路径无效，浪费1 AP并扣分 | 明火封路时应绕行或等待专业救援 |
| 绕路过久 | 王奶奶生命状态下降，评分下降 | 救援要兼顾效率与安全 |
| 火场风险达到100 | 关卡失败，进入复盘 | 风险过高时不应继续深入 |

## 九、胜负条件与评分

### 1. 胜利条件

王奶奶到达安全区，且生命状态大于0。

### 2. 失败条件

- 王奶奶生命状态归零。
- 火场风险值达到100。
- 超过12回合仍未撤离。
- 玩家选择明显违反安全原则的行为，如返回取物。

### 3. 评分结构

满分100分：

| 评分项 | 分值 | 说明 |
| --- | --- | --- |
| 安全原则 | 25 | 判断现场、及时呼救、未做高风险行为 |
| 救援结果 | 25 | 王奶奶成功撤离，生命状态越高得分越高 |
| 沟通安抚 | 20 | 正确安抚、降低恐慌、建立信任 |
| 风险管理 | 20 | 合理使用面罩、观察、清障、关闭燃气 |
| 行动效率 | 10 | 回合数越少得分越高，但不奖励冒险捷径 |

高分并不要求玩家做完所有可选目标。如果为了关闭燃气而导致王奶奶生命状态过低，评分应低于“放弃阀门但快速安全撤离”的路线。

## 十、AI助手与复盘设计

### 1. 关卡中提示

花花提示遵循“三层提示”：

| 层级 | 触发 | 内容 |
| --- | --- | --- |
| 轻提示 | 玩家停顿或首次进入危险区 | 提醒观察、低姿、注意AP |
| 纠错提示 | 玩家选择错误行动 | 说明为什么错，并给出正确方向 |
| 复盘提示 | 失败或通关后 | 总结关键节点和公益知识 |

### 2. AI生成内容边界

AI可生成：

- 王奶奶的背景故事和情绪台词。
- 玩家不同行为下的反馈文案。
- 通关后的个性化复盘。
- 小红花知识卡。

AI不可生成：

- 鼓励玩家冒险进入失控火场的建议。
- 与官方安全原则冲突的救援流程。
- 夸大玩家个人能力、弱化专业救援必要性的文案。

## 十一、开发实现需求

本关需要实现以下系统：

- 10x8网格地图。
- 回合与AP系统，每回合4 AP。
- Fog of War视野解锁。
- 烟雾格、高浓度烟雾格、火焰格。
- 火场风险值与火势扩散。
- 障碍物清除，消耗2 AP。
- 防烟面罩拾取和使用。
- 燃气阀门可选交互。
- 王奶奶生命状态、恐慌值、信任值。
- 王奶奶受限随机移动。
- NPC安抚与跟随撤离。
- 胜负判定与评分。
- AI提示、复盘和知识卡接口。

### NPC跟随实现建议

纯前端网格跟随容易出现卡墙角、重叠、转弯错位等问题。推荐使用队列结构：

1. 维护玩家最近3步的格子坐标队列。
2. 玩家每移动1格，将新坐标入队，旧坐标出队。
3. 王奶奶移动到队列尾部或相邻安全格。
4. 如果王奶奶生命状态低于40，队列延迟增加1步。
5. 加入距离校验，若玩家与王奶奶距离超过2格，提示玩家等待。

## 十二、开发阶段目标

本关作为第二阶段核心关卡，开发目标是验证：

- 4 AP限制下的资源取舍。
- Fog of War对路线判断的影响。
- 清障、面罩、燃气阀门之间的高分取舍。
- NPC恐慌值与信任值是否能形成清晰反馈。
- 护送撤离是否稳定、可读、不卡位。
- 公益安全边界是否被系统规则正确表达。
