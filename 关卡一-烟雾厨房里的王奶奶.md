# 《红花救援队》— 关卡一：烟雾厨房里的王奶奶

## 一、体验模式

游戏只设置一个统一"标准模式"。

统一"标准模式"的设计原则是：

保留游戏挑战，但不做高压惩罚；

默认使用较大按钮和清晰文字，便于全年龄玩家操作；

关卡内提供 AI 助手提示，但不会自动完成操作；

玩家失败后可立即重试，并获得正确流程复盘；

不做复杂语音识别、手势识别和多模式切换，降低开发难度。

## 二、整体玩法执行方式

游戏采用"网格救援 + 急救步骤卡 + 关键 QTE"的组合玩法。

每关的基础执行流程为：

玩家进入关卡场景；

AI 助手"花花"说明当前事件；

玩家在网格地图中移动；

玩家接触 NPC 或道具；

系统弹出可执行行动卡；

玩家选择正确行动；

系统根据选择改变 NPC 状态、时间、评分；

关键急救行为进入 QTE；

完成救援后进入结算；

AI 根据玩家操作生成复盘和知识卡。


## 三、关卡一：烟雾厨房里的王奶奶

### 1. 关卡定位

类型：正式关卡一
时长：约 3-4 分钟
主要目标：救出被困厨房的王奶奶
公益知识点：火灾烟雾逃生、特殊老人沟通
核心机制：网格移动、行动点、浓烟视界(Fog of War)、王奶奶随机移动、清障博弈、烟雾扩散、NPC 恐慌值、安抚沟通、护送撤离


### 2. 场景设置

关卡地图为 8x8 网格。

#### 浓烟视界（Fog of War）

开局时，厨房内部（玩家起点以外区域）全部被浓烟遮挡，显示为"未知（黑色）"状态。玩家移动进入格子或使用"观察"技能（消耗 1 AP）后，该格及相邻格才会解锁可视。这模拟了真实火场中浓烟遮挡视线的情境，防止玩家在第一回合就计算出全局最优解。

#### 清障博弈

在地图中关键路径上设置可清除的障碍物，迫使玩家做资源权衡：

- 防烟面罩所在格子的相邻格有一处[翻倒的冰箱]（清除消耗 2 AP）
- 燃气阀门所在格子的相邻格有一处[倒下的储物架]（清除消耗 2 AP）

玩家必须在"花 AP 清障去拿完美道具"与"跳过道具冒着风险直接救人"之间做选择。

#### 主要元素

| 元素 | 说明 |
| --- | --- |
| 玩家起点 | 厨房门口（开局可见） |
| 王奶奶 | 厨房内部，被烟雾困住，恐慌时会随机移动 |
| 安全区 | 楼道出口 |
| 防烟面罩 | 厨房门口附近，相邻格有[翻倒的冰箱]障碍 |
| 燃气阀门 | 厨房灶台旁，相邻格有[倒下的储物架]障碍 |
| 烟雾格 | 每回合造成伤害，未探索格显示为黑色 |
| 火焰格 | 不可通行 |
| 障碍物 | 翻倒的冰箱、倒下的椅子或杂物，消耗 2 AP 清除 |


### 3. 关卡初始数值

| 项目 | 数值 |
| --- | --- |
| 玩家每回合 AP | 4 |
| 王奶奶生命值 | 100 |
| 王奶奶恐慌值 | 70 |
| 王奶奶信任值 | 20 |
| 烟雾伤害 | 每回合 -5 生命值 |
| 火势扩散 | 每 2 回合扩散 1 格 |
| 王奶奶恐慌移动 | 火势扩散时，随机向相邻格移动 1 格（安抚前） |
| 关卡失败条件 | 王奶奶生命值归零或时间耗尽 |


### 4. 玩家执行流程

#### 第一步：进入厨房区域

开场剧情：

花花："厨房里有浓烟，王奶奶还在里面。她听力不好，也可能因为紧张认不清方向。不要直接拉她，先让她信任你。"

玩家开始于厨房门口。

系统显示：

AP：4

王奶奶生命值：100

王奶奶恐慌值：70

当前回合：第 1 回合

玩家需要先观察地图。厨房内部大部分区域被浓烟遮挡（黑色），玩家可通过移动或"观察"技能（消耗 1 AP）来解锁视野。


#### 第二步：拿取防烟面罩

玩家向防烟面罩所在方向移动。

防烟面罩的相邻格有一处[翻倒的冰箱]障碍物。

玩家面临选择：

- 花 2 AP 清除障碍 → 拿到防烟面罩 → 后续护送烟雾伤害降低 80%
- 不清障直接绕行 → 不拿面罩 → 后续护送压力更大，但省了 AP

如果玩家拿到防烟面罩，面罩提供额外效果：

- 佩戴后玩家视野范围 +1 格（浓烟中能看得更远）
- 后续可给王奶奶使用，降低其烟雾伤害 80%

点击防烟面罩，选择"拾取"。获得道具：防烟面罩。

如果玩家不拿面罩直接接近王奶奶，后续护送时王奶奶会持续受到烟雾伤害。


#### 第三步：关闭燃气阀门

燃气阀门在灶台旁，相邻格有一处[倒下的储物架]障碍物。

玩家可选择是否先清障关闭燃气：

- 花 2 AP 清除储物架 → 关闭燃气（再消耗 1 AP）→ 火势扩散频率降低，通关评分提高
- 不清障不关燃气 → 每 2 回合火势继续扩散，可能封锁撤离路线

玩家面临 AP 资源的三重权衡：**清障拿面罩 → 清障关阀门 → 追奶奶安抚**，三者很难全部做到，必须根据奶奶的随机位置做动态取舍。

这个步骤不是强制，但属于高分操作。


#### 第四步：接近王奶奶

**王奶奶随机移动机制：**

在被安抚之前，每当火势扩散（每 2 回合），王奶奶因恐慌和认知障碍会在厨房区域内随机向相邻格移动 1 格。这打破了玩家的固定路线规划——玩家不能简单地算好路线走过去，而必须根据奶奶的实时位置动态调整 AP 消耗去"拦截"她。奶奶可能跑进烟雾更浓的区域，持续掉血，增加救援紧迫感。

玩家移动到王奶奶相邻格。

触发 NPC 对话：

王奶奶："你是谁啊？我家门在哪里？我看不清……"

系统弹出行动卡：

大声催促她快走

直接拉她离开

蹲下并慢慢说明身份

让她自己找出口

正确选择：蹲下并慢慢说明身份

选择后：

王奶奶恐慌值 -30；

信任值 +30；

解锁"引导撤离"选项。

如果选择"直接拉她离开"：

恐慌值 +30；

王奶奶拒绝移动 1 回合；

评分下降。


#### 第五步：给王奶奶戴防烟面罩

如果玩家已获得防烟面罩，系统弹出道具使用按钮。

玩家点击"使用防烟面罩"。

效果：

王奶奶烟雾伤害降低 80%；

信任值 +10；

评分提高。

如果未使用防烟面罩，王奶奶在烟雾格中每回合持续掉生命值。


#### 第六步：规划撤离路线

王奶奶被安抚后，玩家可以选择"引导撤离"。

执行方式：

王奶奶跟随玩家移动；

玩家每移动 1 格，王奶奶跟随移动 1 格；

如果经过烟雾格，王奶奶掉生命值；

如果经过火焰格，不可通行；

玩家需要绕开火焰和高浓度烟雾。

如果路线被障碍挡住，玩家可消耗 2 AP 清除障碍。


#### 第七步：到达安全区

当玩家和王奶奶都到达安全区时，关卡完成。

系统结算：

是否拿了防烟面罩；

是否关闭燃气；

是否成功安抚；

王奶奶剩余生命值；

用了多少回合；

是否经过高危路线。


### 5. 错误处理

| 错误行为 | 后果 |
| --- | --- |
| 直接拉走王奶奶 | 恐慌值 +30，拒绝移动 1 回合 |
| 不安抚直接撤离 | 无法触发跟随 |
| 不拿防烟面罩 | 烟雾伤害持续，视野无扩展 |
| 不清障放弃面罩 | 后续护送压力增大，但省下 2 AP |
| 不关闭燃气 | 火势扩散更快，奶奶移动更频繁 |
| 不清障放弃阀门 | 火势失控风险增加 |
| 走进火焰格 | 路径无效，浪费 AP |
| 绕路过久 | 王奶奶生命值下降，评分降低 |
| 未及时追上移动的奶奶 | 奶奶持续掉血，救援延迟 |


### 6. 胜利条件

王奶奶成功撤离到安全区，且生命值大于 0。


### 7. 本关开发实现

本关需要实现以下系统：

8x8 网格地图；

浓烟视界（Fog of War）系统；

玩家点击移动；

AP 行动点（每回合 4 AP）；

障碍物清除（消耗 2 AP）；

烟雾伤害；

火势扩散；

王奶奶随机移动（火势扩散时触发）；

道具拾取和使用（面罩含视野扩展）；

NPC 恐慌值与信任值；

NPC 跟随移动；

胜负判定；

结算评分。

#### 技术实现提醒：NPC 跟随机制

纯前端（如 Phaser.js）实现网格跟随时容易出现 Bug（奶奶卡墙角、玩家转弯时重叠等）。推荐使用**队列（Queue）结构**：

1. 维护一个长度为 3 的队列，记录玩家最近 3 步的格子坐标
2. 玩家每走一步，将新坐标入队，旧坐标出队
3. 王奶奶瞬移到队列尾部格子（即延迟 3 步跟随）
4. 加入 `if (distance(player, grandma) > 1)` 拉回判定，防止奶奶掉队

这一方案比每帧寻路更稳定，且天然避免卡墙角问题。

AI 在本关中的使用：

生成王奶奶的背景故事；

生成王奶奶的情绪台词；

根据玩家行为生成不同反馈；

通关后生成复盘文案。
