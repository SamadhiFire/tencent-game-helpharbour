# 《红花救援队》关卡二剧本设定与触发配置：广场黄金四分钟

## 一、文档用途

本文件是关卡二的开发级剧本、地图坐标、事件触发、音频触发、行动卡、物品拾取、NPC台词、错误反馈、评分结算和验收说明。它用于补足正式策划案中未展开的格子范围、动画表现和分支细节。

本文件与以下文档配套使用：

- `关卡二-广场黄金四分钟.md`
- `关卡通用设计规范.md`
- `开发资源清单.md`
- `资源清单/`

本关必须坚持的公益边界：

- 游戏只做急救科普和流程意识训练，不等同于现实急救资质培训。
- 玩家必须先判断现场安全，再检查反应、呼救、取AED、判断呼吸、开始CPR。
- 倒地老人无反应且无正常呼吸后，玩家不能离开老人身边去跑腿；AED必须通过明确指派路人B取得。
- 喂水、扶坐、掐人中、拍视频、围观等行为只作为错误干扰项，不能成为得分捷径。
- AED除颤前必须完成“所有人离开身体”的安全确认；漏掉该步骤时系统必须阻止除颤。
- CPR QTE只表达“持续按压和保持节奏”的参与感，不使用现实时间惩罚玩家阅读和思考。
- 花花可以提示、纠错和复盘，但不能替玩家完成核心判断。

### 5. 事件优先级

同一格、同一回合或同一点击内多个条件同时满足时，按以下顺序处理：

1. 失败判定：生命稳定度归零、严重错误次数达到3、未呼救超时。
2. AED安全判定：未确认离身时禁止除颤。
3. 阶段关键剧情：检查反应、呼救、判断呼吸、开始CPR、AED交接、救护车到达。
4. 交互弹窗：指派路人、行动卡、AED步骤卡。
5. 回合结算：生命稳定度变化、救护车倒计时、路人B移动、群众占格。
6. 音效和弱提示：群众气泡、花花轻提示、目标格闪烁。

### 4. 路人B推荐路径

路人B被指派后，不使用瞬移，必须在网格中真实移动。

| 阶段 | 路径 | 说明 |
| --- | --- | --- |
| 去AED柜 | `(2,5)` -> `(3,5)` -> `(4,5)` -> `(4,4)` -> `(5,4)` -> `(6,4)` -> `(7,4)` -> `(8,4)` | 单程7步，速度2格/回合，约4个救援回合到达 |

动态规则：

- 如果路径下一格被群众占用，路人B本回合最大移动距离从2格降为1格。
- 如果连续2回合被同一群众阻挡，花花强提示玩家使用指挥点疏散通道。
- 群众不会让路人B完全卡死；若堵路超过3回合，系统自动把该群众移到最近空格，但扣现场管理评分。
- 路人B带AED返回时，头顶显示`AED`标签或小图标，避免玩家误以为还没拿到。

## 五、标准流程总览

| 阶段 | 推荐回合 | 玩家目标 | 关键判定 |
| --- | --- | --- | --- |
| S0 现场安全判断 | 回合1 | 先判断广场环境是否安全 | 正确选择“判断现场环境” |
| S1 检查反应 | 回合1 | 靠近老人，轻拍双肩并呼唤 | 确认老人无反应 |
| S2 呼救与AED | 回合1-2 | 指派路人A拨打120，路人B取AED | 救护车倒计时启动，路人B开始跑动 |
| S3 判断呼吸 | 回合2 | 判断无正常呼吸 | 解锁CPR |
| S4 CPR维持 | 救援回合1-7 | 持续CPR，用指挥点维持通道 | QTE结算，路人B往返 |
| S5 AED使用 | AED到达后 | 按顺序完成AED步骤 | 离身确认后才能除颤 |
| S6 继续CPR等待救护 | AED后至倒计时归零 | AED后继续CPR，不提前结束 | 救护车到达时稳定度大于0 |
| S7 结算复盘 | 结束 | 展示评分、知识卡和行为复盘 | 成功或失败 |

推荐高分路线：

1. 开场选择“判断现场环境”。
2. 移动到老人相邻格。
3. 轻拍双肩并呼唤，确认无反应。
4. 明确指派路人A拨打120。
5. 明确指派路人B取AED。
6. 判断呼吸，无正常呼吸后开始CPR。
7. CPR阶段保持节奏，并用指挥点疏散AED通道。
8. AED交接后按步骤操作，除颤后立刻继续CPR。
9. 救护车到达后完成交接。

4. 群众系统判定是否生成、堵路或散开。
5. 刷新指挥点为1/1。

## 八、数值与评分细则

### 1. 生命稳定度模型

| 情况 | 稳定度变化 |
| --- | --- |
| 未检查反应前每回合 | 0，避免开场读题压力 |
| 已确认无反应但未开始CPR | 每玩家回合-10 |
| CPR优秀 | 本救援回合-1 |
| CPR一般 | 本救援回合-3 |
| CPR较差 | 本救援回合-5 |
| 完全漏操作 | 本救援回合-8 |
| 连续2轮优秀 | 额外+2 |
| AED正确使用 | 立即+20 |
| AED后CPR | 每回合消耗在原档位基础上-2，最低0 |
| 错误扶坐 | -15 |
| 错误喂水 | -10 |
| 错误掐人中 | -8 |

### 2. 胜利条件

满足以下全部条件时成功：

- 救护车倒计时归零。
- 老人生命稳定度 >0。
- 玩家已完成呼救。
- 玩家已开始CPR。

### 3. 失败条件

任一条件满足时失败：

- 老人生命稳定度 <=0。
- 检查反应后3个玩家回合仍未拨打120。
- 严重错误次数 >=3。
- AED除颤安全步骤被连续错误尝试3次。

### 4. 评分结构

满分100分：

| 评分项 | 分值 | 细则 |
| --- | --- | --- |
| 急救流程 | 30 | 判断现场8，检查反应8，判断呼吸8，按顺序进入CPR6 |
| 呼救与协作 | 20 | 指派120 10，指派AED 8，指令具体清晰2 |
| CPR表现 | 20 | 平均命中率14，连续优秀4，无长时间中断2 |
| AED使用 | 20 | AED及时到达6，步骤顺序8，离身确认4，除颤后继续CPR2 |
| 现场管理 | 10 | 清理通道4，路人A维持秩序3，未采纳群众错误建议3 |

扣分项：

| 行为 | 扣分 |
| --- | --- |

## 十二、错误处理与教学反馈

| 错误操作 | 游戏后果 | 教学反馈 |
| --- | --- | --- |
| 未判断现场就接近交互 | 阻止关键交互 | “先确认现场安全。” |
| 直接扶起老人 | 稳定度-15 | “不要随意移动无反应者。” |
| 给老人喝水 | 稳定度-10 | “无意识或呼吸异常时不要喂水。” |
| 掐人中 | 稳定度-8 | “偏方不能替代急救流程。” |
| 未拨打120 | 倒计时不启动，3回合后失败 | “专业救援必须尽早在路上。” |
| 未指派AED | AED不会到达，评分上限降低 | “AED越早到达越好。” |
| CPR连续较差 | 稳定度持续下降 | “看准节奏圈，保持稳定。” |
| 群众堵路不处理 | AED延误，现场管理扣分 | “给AED和救援人员留通道。” |
| AED步骤乱序 | 阻止或重试 | “按设备提示顺序操作。” |
| 未离身尝试除颤 | 阻止除颤，错误次数+1 | “必须确认所有人离开身体。” |
| 除颤后停止CPR | 阻止进入等待 | “除颤后继续CPR，直到接手。” |

错误反馈模板：

```text
这一步不建议这样做。
原因：……
现在可以改为：……
```

示例：

```text
这一步不建议这样做。
原因：无反应的人不能随意喂水。
现在可以改为：请旁人拨打120，并判断呼吸。
```

## 十三、失败分支剧本

### 1. 生命稳定度归零

触发条件：老人生命稳定度 `<=0`。

花花台词：

> “救援流程被耽误太久了。下一次先让120在路上，再尽快开始CPR。”

复盘重点：

- 是否没有及时呼救。
- 是否没有开始CPR。
- CPR节奏是否连续较差。
- AED是否被群众堵路延误。

### 2. 未及时拨打120

触发条件：检查反应后3个玩家回合仍未拨打120。

花花台词：

> “现场急救不能只靠一个人。第一时间请旁人拨打120，是关键步骤。”

推荐流程：

> “你，帮我拨打120；你，去取AED；我来检查呼吸并开始CPR。”

### 3. 连续严重错误

触发条件：严重错误次数 `>=3`。

花花台词：

> “这次选择偏离了急救流程。我们重来一次，按顺序完成判断、呼救、AED和CPR。”

复盘重点：

- 哪些选项来自群众错误建议。
- 为什么不能扶坐、喂水或掐人中。
- 为什么AED除颤前要离身。

### 4. AED安全步骤错误

触发条件：AED步骤中连续3次尝试“未离身除颤”或其他高风险乱序。

花花台词：

> “AED必须按提示操作。除颤前确认所有人离开，是不能跳过的安全步骤。”

结果：

- 不表现电击成功。
- 进入AED步骤复盘或从AED步骤第4步重试。

## 十四、成功结算分支

### 1. S级结算

条件建议：

- 总分 >=90。
- 无严重错误。
- 及时拨打120。
- 及时指派AED并完成正确AED步骤。
- CPR平均命中率 >=70%。
- 至少处理1次有效通道问题或无通道延误。

花花台词：

> “你不只做得快，更重要的是顺序正确。真正可靠的救援，是冷静判断和清楚分工。”

知识卡强调：

- 明确指派具体的人。
- 持续CPR，不围观等待。
- AED除颤前必须离身。

### 2. A级结算

条件建议：

- 总分75-89。
- 流程完整。
- 允许少量CPR失误或AED轻微延误。

花花台词：

> “完成得不错。下次可以更早清出AED通道，让设备更快到达。”

### 3. B级结算

条件建议：

- 总分60-74。
- 救护车到达且老人稳定度>0。
- 出现明显错误、AED未使用或CPR表现偏低。

花花台词：

> “你撑到了救援到达，这是重要结果。复盘一下，哪些步骤可以更早、更稳。”

### 4. 失败复盘

花花台词：

> “失败不是结束。我们把流程拆开看：判断现场、检查反应、拨打120、找AED、判断呼吸、CPR、AED、继续CPR。”

复盘面板必须显示：

- 本局最早出错步骤。
- 该步骤正确做法。
- 玩家做对的至少1个行为。
- “再试一次”按钮。

## 十五、AI生成内容边界

AI可生成：

- 围观群众干扰台词，但只能从安全白名单主题中改写。
- 路人A/B协助反馈。
- 倒地老人背景设定，例如“晨练时突然倒地”，但不能替代医疗诊断。
- 通关后的个性化复盘。
- 小红花知识卡文案。

AI不可生成：

- 鼓励喂水、扶起、掐人中等错误行为的正向反馈。
- 具体药物、疾病诊断或治疗承诺。
- 声称玩家已获得急救资质。
- 与AED离身确认冲突的建议。
- 让玩家停止CPR去处理旁事的建议。

推荐AI提示约束：

```text
你是公益游戏《红花救援队》的AI助手花花。
你只能基于预设急救流程做提示：判断现场安全、检查反应、呼救120、取AED、判断呼吸、CPR、AED离身确认、除颤后继续CPR。
不要生成药物、诊断、偏方或替代专业培训的说法。
输出必须短句、亲切、坚定，每次不超过两行。
```

## 十七、开发验收清单

上线关卡二前逐项检查：

- 开场必须自动出现“判断现场环境”行动卡。
- 未完成现场安全判断时，不能检查反应或开始CPR。
- 玩家必须在老人相邻格才能检查反应、判断呼吸和开始CPR。
- 轻拍双肩并呼唤后，老人应显示“无反应”，不能弹出老人对白。
- 指派路人A拨打120后，救护车倒计时必须启动并显示`(0,0)`救护车标志。
- 指派路人B取AED后，路人B必须在网格中真实移动，不得瞬移。
- 路人B移动路径被群众占据时，本回合移动距离应减少或停顿，并有可理解提示。
- CPR阶段玩家不能移动离开老人。
- CPR阶段每轮都有QTE结算、稳定度变化、路人B移动和指挥点刷新。
- QTE命中播放A42，失误播放A43。
- AED柜取出和AED交接均播放A40，并显示AED图标移动。
- AED交接后，道具栏必须显示`18_aed_thumbnail.png`。
- AED步骤卡必须按顺序完成。
- 未确认“所有人离开”时，除颤按钮必须被阻止。
- AED除颤后必须回到CPR，不允许直接等待。
- 救护车倒计时归零播放A45，并进入成功或失败判定。
- 成功结算播放A47，小红花奖励播放A48。
- 所有错误反馈必须说明原因和下一步正确做法。
- 复盘必须显示玩家做对的行为，避免只给失败批评。
- AI生成群众台词不得正向鼓励错误急救行为。
