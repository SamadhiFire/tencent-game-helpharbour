# 《红花救援队》— 关卡二：广场黄金四分钟

## 一、体验模式

游戏只设置一个统一"标准模式"。

统一"标准模式"的设计原则是：

保留游戏挑战，但不做高压惩罚；

默认使用较大按钮和清晰文字，便于全年龄玩家操作；

关卡内提供 AI 助手提示，但不会自动完成操作；

玩家失败后可立即重试，并获得正确流程复盘；

不做复杂语音识别、手势识别和多模式切换，降低开发难度。

## 二、整体玩法执行方式

游戏采用"网格救援 + 急救步骤卡 + 关键 QTE"的组合玩法。

每关的基础执行流程为：

玩家进入关卡场景；

AI 助手"花花"说明当前事件；

玩家在网格地图中移动；

玩家接触 NPC 或道具；

系统弹出可执行行动卡；

玩家选择正确行动；

系统根据选择改变 NPC 状态、时间、评分；

关键急救行为进入 QTE；

完成救援后进入结算；

AI 根据玩家操作生成复盘和知识卡。


## 三、关卡二：广场黄金四分钟

### 1. 关卡定位

类型：正式关卡二
时长：约 3-4 分钟
主要目标：在救护车到来前稳定倒地老人状态
公益知识点：心脏骤停识别、呼救、AED、CPR
核心机制：步骤判断、网格NPC物理移动（跑腿送AED）、回合行为驱动数值、群众占格疏散、CPR 节奏 QTE、AED 步骤卡


### 2. 场景设置

场景为社区广场，地图为 7x7 网格。**网格在本关承担"可视化时间压迫"的职责**：玩家在老人身边站桩做 CPR QTE 的同时，能看到路人 B 在网格上真实移动——每回合啪嗒啪嗒跑 2 格去取 AED，再跑 2 格回来。空间距离感直接转化为"黄金四分钟"的紧张感。

#### 主要元素

| 元素 | 说明 |
| --- | --- |
| 倒地老人 | 关卡核心救助对象，位于地图中央偏左 |
| 玩家 | 初始位于广场入口，接近老人后基本固守其身旁 |
| 路人 A | 位于老人附近，可指派拨打 120 |
| 路人 B | 位于老人附近，可指派去取 AED，在网格上真实移动 |
| AED 柜 | 位于地图右侧边缘（距离老人约 5-6 格） |
| 围观群众 | 会随机出现在老人周围格子上，占格并喊出错误建议 |
| 救护车标志 | 拨打 120 后在地图边缘出现，回合数倒计时 |


### 3. 关卡回合数值模型

**核心原则：数值由回合/行为驱动，不依赖现实时间。** 玩家思考、看弹窗、手速慢不会导致老人白死。

| 项目 | 数值 |
| --- | --- |
| 总回合数 | 10-12 回合 |
| 老人初始生命值 | 100 |
| 每回合基础消耗 | -5 HP（时间流逝不可逆） |
| 未开始 CPR 期间额外消耗 | 每回合额外 -5 HP（合计 -10 HP/回合） |
| CPR 中额外消耗 | 取决于 QTE 成功率（见 QTE 规则） |
| 路人 B 移动速度 | 每回合 2 格 |
| AED 柜距离 | 距老人约 5-6 格（路人 B 往返需 5-6 回合） |
| 错误操作惩罚 | 即时有明确数值扣除（见错误处理） |
| 关卡失败条件 | 老人生命值归零 |


### 4. 玩家执行流程

#### 第一步：靠近倒地老人

玩家移动到倒地老人旁边。

触发事件：

围观群众："他突然倒下了！快来帮帮忙！"

花花提示：

"先确认现场是否安全，再检查意识。"

系统弹出行动卡：

判断现场环境

立刻扶起老人

给老人喝水

掐人中

正确选择：判断现场环境

选择后，进入下一步。


#### 第二步：检查意识

系统弹出行动卡：

轻拍双肩并呼唤

用力摇晃

直接做胸外按压

等他自己醒来

正确选择：轻拍双肩并呼唤

结果：

老人无反应。

花花提示：

"没有反应，需要立即呼救，并请人寻找 AED。"


#### 第三步：指派路人协助

玩家需要点击路人 A。

弹出指令：

请拨打 120

请帮我拍视频

请把老人扶起来

请散开一点

正确选择：请拨打 120

效果：

救护车标志在地图边缘出现，回合倒计时启动（约 10-12 回合）；

评分提高。

玩家再点击路人 B。

弹出指令：

请寻找 AED

请给老人喝水

请围住现场

请不要动

正确选择：请寻找 AED

效果：

路人 B 开始向 AED 柜移动，每回合走 2 格（约 3 回合到达 AED 柜）；

玩家在后续 CPR QTE 过程中，**能在网格地图上肉眼看到路人 B 一格一格向 AED 柜靠近，再一格一格返回**；

这个空间距离感直接转化为玩家的时间压迫——看着路人 B 还在路上跑，老人生命值在掉，就是"黄金四分钟"的体验核心。


#### 第四步：判断呼吸

系统弹出行动卡：

观察胸廓起伏并判断呼吸

直接喂水

把老人扶坐起来

按摩太阳穴

正确选择：观察胸廓起伏并判断呼吸

结果：

系统提示：

"老人无正常呼吸，立即开始 CPR。"


#### 第五步：CPR 节奏 QTE

进入 CPR 界面。画面上出现节奏圆圈，玩家需要按节奏点击。

CPR 期间，屏幕上同时显示网格地图（可看到路人 B 的跑腿进度）和 CPR QTE 区域，无需切换画面。

##### QTE 规则（回合行为驱动）

| 项目 | 规则 |
| --- | --- |
| 每轮 QTE | 对应 1 个回合 |
| 点击节奏 | 按提示圈出现时点击（约 2 秒完成一轮判定） |
| 成功率 ≥ 70% | CPR 优异：本轮生命值 -5（基础）+4（恢复）= 净扣 -1 |
| 成功率 40%-69% | CPR 一般：本轮生命值 -5（基础）+2（恢复）= 净扣 -3 |
| 成功率 ＜ 40% | CPR 较差：本轮生命值 -5（基础）+0 = 净扣 -5，花花提示调整节奏 |
| 连续 2 轮优异 | 老人状态稳定，额外 +2 HP |

##### 生命值回合公式

```
每回合老人 HP 变化 =
  -5（基础消耗，时间流逝不可避免）
  + CPR恢复（≥70%: +4, 40-69%: +2, <40%: +0）
  - 额外惩罚（错误操作，见错误处理）
  + 额外奖励（连续优异、AED 成功使用）
```

玩家**不会因为"手慢/思考"而死**——只有错误决策和 QTE 表现不佳才会恶化局面。CPR 做得好的玩家可以稳定维持生命值（净扣仅 -1），做不好的逐渐下滑。


#### 第六步：AED 到达并使用

**关键时刻：网格交接**

路人 B 在网格上一格一格跑到 AED 柜（约 3 回合），拿到 AED 后再一格一格返回老人位置（约 3 回合）。当路人 B 终于踏入老人相邻格时，触发一个特殊的网格"交接"动画：

- 画面短暂拉回网格视图
- 路人 B 将 AED 递给玩家
- 花花提示："AED 到了！立刻使用！"

这个空间上的"终于到了"瞬间本身就是一种情感释放，比一个倒计时归零的弹窗更有感染力。

系统弹出 AED 步骤卡：

打开 AED

贴上电极片

等待设备分析

所有人离开身体

按提示除颤

继续 CPR

玩家需要按顺序点击。

如果顺序正确：

老人生命值 +20；

后续回合基础消耗从 -5 降为 -3；

进入成功结算。

如果玩家没有派人找 AED：

只能持续 CPR；

通关难度大幅提高；

评分上限降低。


#### 第七步：救护车到达

救护车倒计时结束后，进入结算。

如果老人生命值大于 0，关卡完成。

结算显示：

是否判断环境；

是否及时呼救；

是否寻找 AED；

CPR 准确率；

是否听信错误建议；

最终生命值。


### 5. 群众占格与疏散系统

围观群众不再只是"弹窗"，而是**实体出现在网格上**：

- 每隔 1-2 回合，一名围观群众会随机出现在老人周围的空格上并**占格**
- 占格群众会喊出错误建议（如"快把他扶起来！""给他喝口水吧！"）
- 如果占格群众堵在路人 B 的 AED 运送路径上，路人 B 该回合少走 1 格
- 玩家需要消耗 1 AP 执行"疏散群众"动作，将该格的群众清开

这使"应对围观群众"从"点一下按钮忽略"升级为**需要牺牲 AP 的空间决策**：

- 不清 → 路人 B 的 AED 被堵，送达延迟
- 清 → 消耗 1 AP，但清除了障碍

错误建议台词示例：

"快把他扶起来！"

"给他喝口水吧！"

"掐人中试试！"

"别等救护车了，自己背去医院吧！"

系统不会弹出"忽略"按钮让玩家点。群众台词以气泡形式浮现在占格位置上，玩家需要主动做"疏散"动作来应对。


### 6. 错误处理

| 错误操作 | 后果 |
| --- | --- |
| 直接扶起老人 | 生命值 -15，评分下降 |
| 喂水 | 生命值 -10，评分下降 |
| 未拨打 120 | 救护车不到，无法高分通关 |
| 未寻找 AED | 无 AED 可用，生命值压力极大，评分上限降低 |
| CPR QTE 成功率 <40% | 无恢复加成，老人每回合净扣 -5 |
| AED 步骤错误 | 当回合生命值净扣 -5，花花提示纠正后重试 |
| 未疏散堵路群众 | 路人 B 该回合少走 1 格，AED 送达延迟 |


### 7. 胜利条件

救护车到达时，老人生命值大于 0，且玩家完成基础呼救或 CPR 流程。

高分条件：

判断环境；

检查意识；

及时呼救；

寻找 AED；

CPR QTE 成功率高；

正确使用 AED；

及时疏散堵路群众。


### 8. 本关开发实现

本关需要实现以下系统：

7x7 网格地图（网格在 CPR 期间持续可见，不切换画面）；

玩家点击移动；

NPC 网格物理移动（路人 B 向 AED 柜跑腿，每回合 2 格）；

回合行为驱动的生命值模型（不依赖现实时间）；

救护车回合倒计时；

CPR 节奏 QTE（与网格地图同屏显示）；

AED 交接动画（路人 B 踏入相邻格触发）；

AED 步骤卡（按顺序点击）；

结算评分。

AI 在本关中的使用：

生成围观群众的干扰台词；

生成倒地老人背景；

根据玩家操作生成赛后复盘；

生成本关小红花知识卡。
