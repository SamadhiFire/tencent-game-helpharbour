## 设计规范

| 项目 | 规格 |
|------|------|
| 画布尺寸 | 64×64 px（UI头像尺寸） |

## UI使用说明

- **头像尺寸**：64×64 px，用于对话面板、提示弹窗等UI元素
- **表情切换**：根据游戏流程自动切换——正常态为默认，提示态用于急救知识引导，鼓励态用于玩家尝试操作时，欣慰态用于成功完成急救步骤后
- **动画建议**：表情切换时可叠加缩放/弹跳微动效（代码实现）

---

## 游戏内使用

在UI面板中直接作为 64×64 头像图标使用。代码引用示例：

```js
const huahuaAvatars = {
  normal:    '02_huahua_normal.png',
  hint:      '02_huahua_hint.png',
  encourage: '02_huahua_encourage.png',
  relieved:  '02_huahua_relieved.png',
};
```
