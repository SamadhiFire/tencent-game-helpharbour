# 王奶奶 角色精灵

> 资源编号：03 | 关卡：关卡一（烟雾厨房里的王奶奶） | 优先级：P0

---

## 设计规范

| 项目 | 规格 |
|------|------|
| 画布尺寸 | 64×64 px（游戏格子大小，角色本体 32×32） |
| 角色本体 | ~32×32 px，居中放置于 64×64 格子内 |
| 格式 | PNG（透明背景） |
| 风格 | Q版像素艺术，1px黑色描边，暖色调 |
| 头身比 | ~1:1 大头Q版，略驼背老态 |
| 配色 | 银灰色发髻（深色发簪） + 深蓝色碎花棉袄 + 深灰色裤子 + 黑色布鞋 + 暗色眼珠 |

| 文件名 | 内容说明 |

| 文件名 | 内容说明 |
|--------|----------|

## 动画播放说明

- **恐慌颤抖**：panic_1 → panic_2 交替循环播放，建议帧间隔 ~80ms（快速颤抖感）
- **安抚态**：从恐慌态过渡到安抚态时，可渐变或直接切换，静态1帧
- **跟随移动**：walk_1 → walk_2 交替循环播放，建议帧间隔 ~180ms（老人步速较慢）
- **站立**：静态1帧，可叠加微小呼吸晃动

## 游戏内使用

王奶奶为关卡一的核心被救援对象，角色精灵放置于 64×64 格子内（角色本体 32×32，居中），需额外偏移使角色底部对齐格子底边。代码引用示例：

```js
const grandmaSprites = {
  idle:    '03_grandma_idle.png',
  panic:   ['03_grandma_panic_1.png', '03_grandma_panic_2.png'],
  calm:    '03_grandma_calm.png',
  walk:    ['03_grandma_walk_1.png', '03_grandma_walk_2.png'],
};
```
