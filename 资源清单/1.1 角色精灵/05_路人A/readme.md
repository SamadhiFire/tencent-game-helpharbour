## 设计规范

| 项目 | 规格 |
|------|------|
| 画布尺寸 | 64×64 px（游戏格子大小，角色本体 32×32） |

## 动画播放说明

- **站立**：静态1帧，路人初始待命状态
- **拨打电话**：call_1 → call_2 按序播放，call_1 播放约 300ms（拨号阶段），call_2 持续播放至通话结束

## 游戏内使用

路人A为关卡二的辅助角色，在广场场景中发现倒地老人后拨打120急救电话。角色本体 32×32 居中放置于 64×64 格子内。代码引用示例：

```js
const bystanderSprites = {
  idle:  '05_bystander_idle.png',
  call:  ['05_bystander_call_1.png', '05_bystander_call_2.png'],
};
```
