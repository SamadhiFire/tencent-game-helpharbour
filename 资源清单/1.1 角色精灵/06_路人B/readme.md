## 设计规范

| 项目 | 规格 |
|------|------|
| 画布尺寸 | 64×64 px（游戏格子大小，角色本体 32×32） |
| 角色本体 | ~32×32 px，居中放置于 64×64 格子内 |
| 格式 | PNG（透明背景） |
| 风格 | Q版像素艺术，1px黑色描边，暖色调 |
| 头身比 | ~1:1 大头Q版 |
| 配色 | 棕色马尾辫（发带） + 珊瑚红色休闲T恤 + 米色卡其裤 + 棕色平底鞋 + 小挎包 |
| ⚠️ 重要 | 必须有4方向跑步移动动画 |

## 动画播放说明

- **站立**：静态1帧，路人初始待命状态
- **跑步动画**：每方向2帧交替循环播放，建议帧间隔 ~120ms（跑步速度比走路快，帧间隔更短）
- 跑步姿态与走路不同：身体明显前倾，手臂大幅摆动，马尾辫和挎包随跑动弹跳

## 游戏内使用

路人B为关卡二的辅助角色，在广场场景中奔跑疏散人群或寻找帮助。角色本体 32×32 居中放置于 64×64 格子内。代码引用示例：

```js
const bystanderBSprites = {
  idle:  '06_bystanderB_idle.png',
  run: {
    down:  ['06_bystanderB_run_down_1.png',  '06_bystanderB_run_down_2.png'],
    up:    ['06_bystanderB_run_up_1.png',    '06_bystanderB_run_up_2.png'],
    left:  ['06_bystanderB_run_left_1.png',  '06_bystanderB_run_left_2.png'],
    right: ['06_bystanderB_run_right_1.png', '06_bystanderB_run_right_2.png'],
  },
};
```
