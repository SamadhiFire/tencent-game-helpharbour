## 游戏内使用

围观群众为关卡二的场景填充NPC，散布在广场中围观突发事件。4个变体可随机混合放置，增加场景丰富度。角色本体 32×32 居中放置于 64×64 格子内。代码引用示例：

```js
const crowdSprites = [
  '07_crowd_variantA_green.png',
  '07_crowd_variantB_purple.png',
  '07_crowd_variantC_yellow.png',
  '07_crowd_variantD_blue.png',
];

// 随机选取变体填充场景
function getRandomCrowd() {
  return crowdSprites[Math.floor(Math.random() * crowdSprites.length)];
}
```
