# 12 防烟面罩 道具图标

> 资源编号：12 | 关卡：关卡一（烟雾厨房里的王奶奶） | 优先级：P0

---

## 设计规范

| 项目 | 规格 |
|------|------|
| 格式 | PNG（透明背景） |
| 风格 | Q版像素艺术，1px黑色描边，暖色调 |
| 用途 | 可拾取物件，关卡一厨房场景中可捡取穿戴 |

---

## 图标清单

| 文件名 | 用途 | 内容说明 |
|--------|------|----------|
| `12_mask_grid.png` | 格子内图标 | 防烟面罩正面视图，亮橙红色工业面罩+透明大护目镜+两侧白色滤毒罐+顶部黑色调节带，放在网格格子内供玩家拾取 |
| `12_mask_thumbnail.png` | 道具栏缩略图 | 防烟面罩俯视简化缩略图，紧凑设计适合底部道具栏小图标槽位 |

---

## 游戏内使用

- **格子内图标**：放置在厨房地图网格中，玩家走到该格即可拾取
- **道具栏缩略图**：拾取后在底部道具栏显示，玩家可点击使用

```js
const maskSprites = {
  grid:      '12_mask_grid.png',
  thumbnail: '12_mask_thumbnail.png',
};
```
