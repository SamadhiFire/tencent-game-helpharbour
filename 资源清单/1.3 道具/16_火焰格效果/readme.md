## 设计规范

| 项目 | 规格 |
|------|------|
| 格式 | PNG（透明背景） |
| 风格 | Q版像素艺术，暖色调火焰效果 |
| 用途 | 不可通行警告格子效果，关卡一厨房火焰区域 |

## 动画帧清单

| 文件名 | 内容说明 |
|--------|----------|
| `16_fire_frame1.png` | 火焰帧1：橙红色火焰从格子中心升起，顶部亮黄白色焰尖，底部暗红炭火基座，小火星向上飘散 |
| `16_fire_frame2.png` | 火焰帧2：火焰形态比帧1略高略宽，火焰卷曲舞动形态不同，交替循环产生闪烁动画效果 |

## 动画播放说明

- **火焰动画**：frame1 → frame2 交替循环播放，建议帧间隔 ~150ms
