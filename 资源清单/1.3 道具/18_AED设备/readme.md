## 游戏内使用

- **格子内图标**：放置在广场地图网格中或从AED柜中取出，玩家走到该格拾取
- **道具栏缩略图**：拾取后在底部道具栏显示，玩家可点击使用

```js
const aedSprites = {
  grid:      '18_aed_grid.png',
  thumbnail: '18_aed_thumbnail.png',
};
```
